# 🏨 Hotel Booking System

A modern and user-friendly Hotel Booking System that allows users to search, view, and book hotel rooms online. The platform provides a seamless booking experience with room details, availability tracking, and reservation management.

## 🚀 Features

### 👤 User Features
- User Registration & Login
- Browse Available Rooms
- View Room Details & Amenities
- Search Hotels by Location
- Book Rooms Online
- View Booking History
- Responsive Design

### 🛠️ Admin Features
- Admin Dashboard
- Manage Hotels & Rooms
- Add, Update, and Delete Room Details
- Manage User Bookings
- View Reservation Records
- Monitor Room Availability

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/subodh182/Hotel-booking-system.git
```

### 2. Navigate to Project Directory

```bash
cd Hotel-booking-system
```

### 3. Install Dependencies

```bash
npm install
```

or

```bash
mvn install
```

### 4. Configure Database

Create a MySQL database and update the database credentials in the configuration file.

### 5. Run the Application

```bash
npm start
```

or

```bash
mvn spring-boot:run
```
