# Hotel Booking System

A web-based hotel booking system.

## Tech Stack

- HTML
- JavaScript

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/subodh182/Hotel-booking-system.git
   ```
2. Open the project in your preferred editor.
3. Open `index.html` in a browser to run the app.

## Features

- Browse available rooms
- Book rooms online
- Simple front-end interface
